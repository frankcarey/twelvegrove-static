Mid-Hudson Library System

MHLS is a regional New York library system serving member libraries from Westchester to Albany. They needed a way to organize their materials and resources in a way that would be user friendly for both their patrons AND their staff and administrators.

Twelve Grove built a comprehensive Content Management System that included a MS Word like editor, advanced image handling and dynamic resizing, content tagging, custom administration fields, and much more. Even with the advanced features on the backend, the interface remained easy to use straight forward to administer.
